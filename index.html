Pertanyaan yang Sering Diajukan (FAQ)

Temukan jawaban untuk pertanyaan umum seputar layanan Seraphic.Std

  1. Chat WA: Hubungi kami via WhatsApp untuk konsultasi
  2. Sepakat + DP: Deal harga dan bayar DP sesuai ketentuan
  3. Proses & Revisi: Desain dikerjakan, revisi sampai puas (Maksimal 3x Revisi)
  4. Lunas + File Dikirim: Bayar sisa, file langsung dikirim

Besaran DP (Down Payment) adalah 50% di awal sebelum proses desain dimulai. Sisa pembayaran dilakukan setelah desain disetujui dan sebelum file final dikirimkan.

Kami menerima berbagai metode pembayaran untuk kemudahan Anda, di antaranya: Transfer Bank (BCA, BRI, Mandiri, dll), E-Wallet (DANA, GoPay, OVO, ShopeePay), dan juga QRIS.

Waktu pengerjaan bervariasi tergantung tingkat kesulitan dan antrean. Rata-rata pengerjaan memakan waktu 1-3 hari kerja setelah DP diterima dan brief disepakati.

Tentu bisa. Kami membedakan antara revisi minor dan revisi konsep (mayor):

  • Revisi Minor: Perubahan kecil seperti warna, font, atau posisi elemen. Kami memberikan maksimal 3x revisi minor gratis.
  • Revisi Konsep: Perubahan total konsep atau ide dasar dari brief awal yang telah disepakati. Ini sering dihitung sebagai pembuatan ulang desain dan akan dikenakan biaya tambahan.

Tentu, setelah pelunasan Anda akan mendapatkan file siap cetak beresolusi tinggi (PNG/JPG) beserta file mentah/source file (AI/PSD/CDR/EPS) sesuai kesepakatan awal.

Solusi desain premium dan kreatif. Kami mempercepat visi brand Anda dengan keindahan dan kualitas.
